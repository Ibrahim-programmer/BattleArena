# BattleArena
A console-based battle game developed in Java using Object-Oriented Programming (OOP) concepts such as inheritance, abstraction, and encapsulation.

## Features

* 2 Player character classes:

  * Warrior
  * Mage
  * Archer
* Randomized attack damage system
* Turn-based battle mechanics
* Health management system
* Winner detection
* Console-based interactive gameplay

## Technologies Used

* Java
* OOP Principles
* Scanner Class
* Math.random()
